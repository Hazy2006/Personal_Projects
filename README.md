A secure password generator with strength analysis.

**Features:**
- Cryptographically secure password generation
- Customizable length and character sets
- Password strength calculator
- Security best practices

**Run:** `python password_generator/password_gen.py`

[View Details](password_generator/README.md)

### 4. Weather CLI
Real-time weather information from OpenWeatherMap API.

**Features:**
- Live weather data for any city
- Multiple temperature units (Celsius/Fahrenheit)
- Detailed weather information
- Clean, formatted output

**Run:** `python weather_cli/weather.py`

**Note:** Requires OpenWeatherMap API key (see [setup instructions](weather_cli/README.md))
